Pour installer le projet : 
Placez vous à la racine du projet, tapez pip install -r requirements.txt
Optionnel : Placez vous ensuite dans le dossier frontend et tapez npm install
Pour lancer le serveur, placez vous dans votre environnement virtuel avec source venv/bin/activate, puis pip install -r requirements.txt, puis : python manage.py runserver
Le serveur peut être long (quelques minutes) à démarrer.
Pour le compilateur javascript il est automatiquement configuré, après avoir fait un npm install (la première fois) placez vous dans le dossier frontend avec une console et tapez npm run dev
